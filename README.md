🍦 Melting Moments – Ice Cream Website

📌 Project Overview
**Melting Moments** is a responsive and visually appealing ice cream website developed using **HTML5, CSS3, and Bootstrap 5**.  
This project showcases an attractive landing page for an ice cream brand with stylish navigation, beautiful background images, and a clean user-friendly layout.
The website is designed to highlight the brand identity and provide users with easy navigation to different sections such as **Home, About, Menu, and Contact Us**.

🚀 Features
- Responsive multi-page website design
- Attractive navbar with hover effects
- Beautiful ice cream themed background image
- Stylish typography using Google Fonts
- Clean and simple user interface
- Mobile-friendly layout using Bootstrap
- Smooth hover effects on navigation links
  
🛠️ Technologies Used
- HTML5
- CSS3
- Bootstrap 5.3
- Google Fonts
  
📂 Project Structure
- `index.html` – Home Page
- `about.html` – About Page
- `menu.html` – Menu Page
- `contact.html` – Contact Us Page
  
🎯 Purpose of the Project
This project was created to practice and improve my **frontend web development skills** by building a real-world themed website.  
It helped me gain hands-on experience in:
- Creating responsive layouts
- Designing navigation bars
- Using Bootstrap components
- Applying custom CSS styling
- Building multi-page websites
  
💡 UI Highlights
- Eye-catching ice cream theme
- Elegant brand title: **Melting Moments**
- Interactive navigation menu
- Full-screen background image
- Smooth and modern visual appearance
